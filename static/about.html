About Geoboner

Geoboner is a platform for hosting and sharing information about your boners. You can share location for each new boner, or search recent and nearby boners.

Geoboner's backend is implemented in OCaml (using the Opium library for the web server) and is licensed under the GNU General Public License (Version 3).

Inspiration for Geoboner comes from this post on /g/.

You can find the source code and project page on Github. Feel free to file bugs, request improvements, or contribute changes through a pull request.
